My Box Model Reference

The Box Model refers to a box that wraps around an HTML element that controls the design and layout of the element.

CSS Property Description Example
width sets the width of an element's box width: 200px;
min-width ensures a minimum width of an element's box min-width: 200px;
max-width ensures the maximum width of an element's box max-width: 250px;
 
height sets the height of an element's box height: 200px;
min-height ensures the minimum height of an element's box min-height: 200px;
max-height ensures the maximum height of an element's box max-height: 200px;
 
padding sets the amount space between the content area and the border, can be set using:
  • 1 value to set all padding
  • 2 values; first to set vertical (top + bottom) padding and the second to set horizonal (left + right) padding
  • 3 values; first to set top padding, second to set horizontal padding and third to set bottom padding
  • 4 values; first to set top padding, second to set right padding, third to set bottom padding and fourth to set left padding
 padding: 10px;
padding: 10px 15px;
padding: 5px 15px 10px;
padding: 5px 15px 10px 10px;
padding-top sets the amount space between the top of the content area and the top border padding-top: 5px;
padding-right sets the amount space between the right of the content area and the right border padding-right: 15px;
padding-bottom sets the amount space between the bottom of the content area and the bottom border padding-bottom: 10px;
padding-left sets the amount space between the left of the content area and the left border padding-left: 10px;
 
border adds a border surrounding the content area and padding of an element by specifying the thickness, style and colour border: 1px solid black;
border-top adds a top border surrounding the top of the content area and top padding of an element by specifying the thickness, style and colour border-top: 1px dashed grey;
border-right adds a right border surrounding the right of the content area and right padding of an element by specifying the thickness, style and colour border-right: 2px dotted blue;
border-bottom adds a bottom border surrounding the bottom of the content area and bottom padding of an element by specifying the thickness, style and colour border-bottom: 3px solid red;
border-left adds a left border surrounding the left of the content area and left padding of an element by specifying the thickness, style and colour border-left: 4px dotted green;
border-radius used to modify the corners of an element's border box, makes it curved as apposed to straight border-radius: 5%;
 
margin sets the amount of space between the border and the outside edge of an element, can be set using:
  • 1 value to set all margin
  • 2 values; first to set vertical (top + bottom) margins and the second to set horizonal (left + right) margins
  • 3 values; first to set top margin, second to set horizontal margins and third to set bottom margin
  • 4 values; first to set top margin, second to set right margin, third to set bottom margin and fourth to set left margin
  • auto to horizonally center an element within its container
 margin: 10px;
margin: 10px 15px;
margin: 5px 15px 10px;
margin: 5px 15px 10px 10px;
margin: 0 auto;
margin-top sets the amount of space between the top border and the top outside edge of an element margin-top: 5px;
margin-right sets the amount of space between the right border and the right outside edge of an element margin-right: 15px;
margin-bottom sets the amount of space between the bottom border and the bottom outside edge of an element margin-bottom: 10px;
margin-left sets the amount of space between the left border and the left outside edge of an element margin-left: 10px;
 
overflow controls what happens to content that overflows outside its box, can be set using:
  • visible - overflowing content will show outside of element's box
  • hidden - overflowing content will be hidden
  • scroll - adds scroll bars to access overflowing content
 overflow: visible;
overflow: hidden;
overflow: scroll;
visibility can be used to set the visibility of an element, without removing the element from the page visibility: hidden;
box-sizing controls which aspect of the box is determined by the height and width properties, can be set using:
  • content-box - default value, actual size includes the content box
  • border-box - actual size includes the content box, paddings and borders
 box-sizing: content-box;
box-sizing: border-box;
